# Zip Zap TV
<p align="justify">
  Small project done in Pico-8 over the course of a weekend for the Art and Play 2025 Pombal Game Jam. The theme was "TV Game Show", so we approached it in a WarioWare fashion, where you keep zapping through 4 TV channels, each with a minigame that carries progress, with the challenge coming from having to complete all 4, but in an asynchronous manner.
</p>

## Build/Run Instructions
Just load and run <code>game.p8</code> in Pico-8 and you're good to go. It's easier to edit the source files in an external text editor, just be sure to update <code>game.p8</code> with <code>game.lua</code> after any changes.
